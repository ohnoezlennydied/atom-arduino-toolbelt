# Arduino-toolbelt

Arduino toolbelt on Atom with official CLI tool.

## System requirements
### Must
- OS X (other OS is not supported now)
- Arduino IDE (ver >=1.5.0)

### Reccomend
- Latest version OS
- Latest version Atom

## Usage
1. Download Arduino IDE from Arduino website. https://www.arduino.cc
2. Modify this package config.
